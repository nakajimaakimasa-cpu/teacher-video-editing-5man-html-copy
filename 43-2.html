2本目のサンプル動画を企画・編集する

1本目とは異なるスタイルで、あなたの多様な編集スキルを見せる2本目を作ります。

今回はビジネス系の動画（約1分）を編集していきます。ステップごとに進めていけば、プロっぽい動画が完成します。

ステップ1：編集の準備

Premiere Proを起動して、編集の準備をします。

プロジェクトを作成する

  1. Premiere Proを起動

  2. 「新規プロジェクト」をクリック

  3. プロジェクト名を「サンプル動画2_ビジネス系」に設定

  4. 保存場所を選んで「作成」をクリック

素材を読み込む

  1. プロジェクトパネルをダブルクリック

  2. 素材フォルダから以下を選択して読み込み

    • 動画素材2（ダウンロード済みのもの）
    • BGM
    • 効果音
    • 画像素材

タイムラインに配置する

  1. 動画素材をプロジェクトパネルからタイムラインにドラッグ

  2. V1トラックに配置されていることを確認

これで編集の準備は完了です。

ステップ2：色調補正

カット編集の前に、まず映像全体の色味を整えます。

先に色調補正をする理由は、後でネスト化してからカットするためです。

調整レイヤーを作成する

  1. プロジェクトパネルを右クリック

  2. 「新規項目」→「調整レイヤー」を選択

  3. 設定はそのままで「OK」をクリック

  4. 作成した調整レイヤーをタイムラインのV2トラックに配置

  5. 調整レイヤーの長さを動画全体に合わせて伸ばす

Lumetriカラーで色を調整する

  1. 調整レイヤーを選択した状態で、「ウィンドウ」→「Lumetriカラー」を開く

  2. 「基本補正」セクションの「自動」ボタンをクリック

  3. 自動補正をベースに、必要に応じて微調整

    • 露光量：映像が暗ければ少し上げる
    • コントラスト：メリハリをつけたければ調整

🔍 調整のコツ

まずは「自動」を押して、そこから微調整するのが効率的です。

やりすぎると不自然になるので、自動補正のままでもOKです。

ステップ3：ネスト化

色調補正した映像をネスト化します。手順は1本目と同じです。

ネスト化の手順

  1. タイムライン上の動画クリップ（V1）と調整レイヤー（V2）を両方選択

    Shiftキーを押しながらクリックで複数選択できます

  2. 右クリック→「ネスト」を選択

  3. ネスト名を入力（例：「ビジネス系_本編」）

  4. 「OK」をクリック

これで動画と調整レイヤーが1つのクリップにまとまりました。

ステップ4：カット

ネスト化したクリップの不要部分を削除していきます。手順は1本目と同じです。

カットする箇所

  • 言い間違い
  • 「えーと」「あのー」などの言いよどみ
  • 不自然な間（沈黙）
  • パートとパートの間の余白

カットのタイミング目安

カット位置 タイミング 備考
言い始め 発声の直前（0フレーム） ここはギリギリでカット
言い終わり 発声終了から2フレーム後 違和感があれば調整OK

🔍 調整のコツ

言い始めは必ず0フレーム（ギリギリ）でカットしましょう。

言い終わりは2フレームが目安ですが、違和感があれば微調整してください。

ステップ5：テロップ

喋っている内容に合わせてテロップを入れていきます。

テロップの基本設定

  • フォント：基本テロップ１
  • サイズ：100
  • 文字は太く、見やすく設定
  • 配置：画面下部に統一

テロップの入れ方

  1. ツールパネルから「文字ツール」を選択

  2. プログラムモニター上でクリックしてテキストを入力

  3. プロパティパネルでフォント・サイズを調整

  4. テロップの長さを喋っているタイミングに合わせる

ステップ6：テロップの強調

重要な部分はテロップスタイルや色を変えて目立たせます。

使用するテロップスタイル

スタイル名 使う場面
ポジティブテロップ2 冒頭のフック（「やばいこと言いますが」）
ネガティブテロップ7 ネガティブな内容（「YouTubeで再生数伸びないです」）
ポジティブテロップ5 強い訴求（「本質のSNSマネタイズ」）
ポジティブテロップ1 実績・数字（「総フォロワー25万人」）
黄色文字 重要キーワード（下記参照）

黄色文字にする箇所（重要キーワード）

  1. 「4つの機能」

  2. 「4つチェック」

  3. 「YouTubeショートのリミックス機能」

  4. 「発見欄」

  5. 「プレミア公開」

  6. 「コメントや再生数が」

  7. 「アルゴリズム」

  8. 「YouTubeスタジオのサムネテスト」

  9. 「サムネイル」

  10. 「クリック率」（「1番クリック率」の部分）

  11. 「クリック率が安定して」

  12. 「コミュニティ投稿」

  13. 「視聴者とつながれる」

  14. 「次の企画のヒント」

  15. 「下のボタン」

黄色文字の設定方法

  1. 色を変えたい文字をドラッグして選択

  2. プロパティパネルの「塗り」で黄色を選択

ステップ7：サイドテロップ

画面左側にセクションごとのタイトルを表示するサイドテロップを作ります。作り方は1本目と同じです。

セクション一覧

以下のセクションごとにサイドテロップを作成します。

  1. YouTubeショートのリミックス機能

  2. プレミア公開

  3. YouTubeスタジオのサムネテスト

  4. コミュニティ投稿

  5. 終わりに

サイドテロップの作り方

  1. ツールパネルから「長方形ツール」を選択

  2. 画面左側に黒い長方形を描画（背景用）

  3. プロパティパネルで「ストローク」にチェックを入れて白枠を追加

  4. ストロークの幅を調整（2〜4px程度）

  5. 「文字ツール」でセクション名を入力

  6. 文字色を白に設定

  7. シェイプの中央にテロップを配置

サイドテロップをコピーして使い回す

  1. 1つ目のサイドテロップができたら、選択してCtrl + C（Mac：Cmd + C）でコピー

  2. Ctrl + V（Mac：Cmd + V）でペースト

  3. テキストをダブルクリックして、セクション名を変更

  4. 該当するセクションのタイミングに配置

ステップ8：画像挿入・シェイプ作り

動画の内容を視覚的に補強するために、画像とシェイプを追加します。

※画像素材は事前にダウンロードしておいてください。

画像を挿入する箇所

箇所（台本の該当部分） 挿入するもの
「ショート動画でバズっている音源をそのまま使える機能で」 画像
「ライブ配信のように同時視聴機能で」 画像
「公開直後からコメントや再生数が集まりやすいため」 画像
「複数のサムネイルを同時にテストして」 画像
4つ目（コミュニティ投稿）のセクション シェイプを作成

画像の挿入方法

  1. プロジェクトパネルから画像素材をタイムラインにドラッグ

  2. 該当する箇所のタイミングに配置

  3. プロパティパネルで位置・サイズを調整

シェイプの作り方（4つ目のセクション用）

1本目と同じ手順で、白い半透明のパネルを作成します。

  1. ツールパネルから「長方形ツール」を選択

  2. 画面左側にドラッグして長方形を描画

  3. プロパティパネルの「塗り」をクリックして白を選択

  4. 「不透明度」スライダーで半透明に調整（50〜70%程度）

  5. 「角丸の半径」に数値を入力して角を丸くする

  6. 「文字ツール」でテキストを入力してシェイプの中央に配置

ステップ9：アニメーション

テロップや画像にアニメーションをつけて、動きのある動画にします。

冒頭のフック部分

箇所 アニメーション
「YouTubeで再生数が伸びない」 画像を白黒 + 画角80%縮小

サイドテロップ（1〜4つ目）

箇所 アニメーション
YouTubeショートのリミックス機能 上からペコ
プレミア公開 上からペコ
YouTubeスタジオのサムネテスト 上からペコ
コミュニティ投稿 上からペコ

画像のアニメーション

箇所 アニメーション
1枚目の画像（ショート動画でバズっている音源〜） 上から下 or 左から右に動かす
「ライブ配信のように同時視聴機能で」の画像 退場時：右から左へ退場
「複数のサムネイルを同時にテストして」の画像 背景をブラーにする

その他のアニメーション

箇所 アニメーション
「視聴者とつながれる」 上から登場
「総フォロワー25万人」 画角アップ + 文字サイズ大きく

🔍 アニメーションのつけ方

具体的なアニメーションのつけ方は動画で解説しています。

動画を見ながら一緒に作っていきましょう。

ステップ10：BGM

動画全体にBGMを追加します。

BGMの配置

  1. 用意したBGMをタイムラインのA2トラックにドラッグ

  2. 動画全体の長さに合わせて配置

フェードイン・フェードアウト

必要に応じて、BGMの最初と最後にフェードをつけましょう。

ステップ11：SE

ポイントとなる箇所に効果音を追加します。

SEをつける箇所

  • サイドテロップの切り替わり
  • シェイプの入場
  • 強調したいポイント

SEの基本ルール

同じジャンルの情報には同じSEを使って統一感を出しましょう。

ステップ12：音量調整

声、BGM、SEのバランスを整えます。

音量の目安

素材 音量目安
声 -6dB
BGM -25dB
SE -18dB

🔍 調整のコツ

声がしっかり聞こえることが最優先です。

BGMは「薄く流れている」くらいがちょうどいいです。

ステップ13：最終確認

書き出す前に、全体を通してチェックしましょう。

確認チェックリスト

  • 動画全体が約1分に収まっているか
  • テロップに誤字脱字がないか
  • 音量バランスは適切か（声がしっかり聞こえるか）
  • カットのつなぎが不自然じゃないか
  • アニメーションのタイミングがズレていないか
  • 1本目と見比べて違いが明確か

🔍 確認のコツ

一度最初から最後まで通して再生してみましょう。

気になるところがあればメモして、まとめて修正すると効率的です。

ステップ14：書き出し

確認が終わったら、動画を書き出します。

書き出しの手順

  1. 「ファイル」→「書き出し」→「メディア」を選択

  2. 形式：H.264を選択

  3. プリセット：YouTube 1080p HDを選択

  4. 出力名をクリックして、保存場所とファイル名を設定

  5. 「書き出し」をクリック

ステップ15：YouTubeにアップロード

完成した動画をYouTubeにアップロードします。

アップロードの手順

  1. YouTubeにログイン

  2. 右上の「作成」ボタン→「動画をアップロード」をクリック

  3. 書き出した動画ファイルを選択

  4. タイトル・説明文を入力

  5. 公開設定を「限定公開」に設定（ポートフォリオ用のため）

  6. 「公開」をクリック

🔍 ポイント

ポートフォリオ用なので「限定公開」にしておくと、リンクを知っている人だけが視聴できます。
