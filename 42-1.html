1本目のサンプル動画を編集する

撮影した素材を使って、自己紹介ポートフォリオを完成させましょう。

ステップごとに進めていけば、プロっぽい動画が完成します。

サンプル動画を確認する

まずは完成形のサンプル動画をもう一度見てみましょう。

  • 顔出しありバージョン
  • 顔出しなしバージョン

ステップ1：編集の準備

Premiere Proを起動して、編集の準備をします。

プロジェクトを作成する

  1. Premiere Proを起動

  2. 「新規プロジェクト」をクリック

  3. プロジェクト名を「自己紹介ポートフォリオ」に設定

  4. 保存場所を選んで「作成」をクリック

素材を読み込む

  1. プロジェクトパネルをダブルクリック

  2. 素材フォルダから以下を選択して読み込み

  • 撮影した動画ファイル

  • BGM

  • 効果音

  • 画像素材

タイムラインに配置する

  1. 撮影した動画をプロジェクトパネルからタイムラインにドラッグ

  2. V1トラックに配置されていることを確認

これで編集の準備は完了です。

ステップ2：色調補正

カット編集の前に、まず映像全体の色味を整えます。

先に色調補正をする理由は、後でネスト化してからカットするためです。

調整レイヤーを作成する

  1. プロジェクトパネルを右クリック

  2. 「新規項目」→「調整レイヤー」を選択

  3. 設定はそのままで「OK」をクリック

  4. 作成した調整レイヤーをタイムラインのV2トラックに配置

  5. 調整レイヤーの長さを動画全体に合わせて伸ばす

Lumetriカラーで色を調整する

  1. 調整レイヤーを選択した状態で、「ウィンドウ」→「Lumetriカラー」を開く

  2. 「基本補正」セクションの「自動」ボタンをクリック

  3. 自動補正をベースに、必要に応じて微調整

  • 露光量：映像が暗ければ少し上げる

  • コントラスト：メリハリをつけたければ調整

🔍 調整のコツ

まずは「自動」を押して、そこから微調整するのが効率的です。

やりすぎると不自然になるので、自動補正のままでもOKです。

ステップ3：ネスト化

色調補正した映像をネスト化します。手順は1本目と同じです。

ネスト化の手順

  1. タイムライン上の動画クリップ（V1）と調整レイヤー（V2）を両方選択

    Shiftキーを押しながらクリックで複数選択できます

  2. 右クリック→「ネスト」を選択

  3. ネスト名を入力（例：「ビジネス系_本編」）

  4. 「OK」をクリック

これで動画と調整レイヤーが1つのクリップにまとまりました。

ステップ4：カット

ネスト化したクリップの不要部分を削除していきます。

台本通りに撮影していれば、カット後は60秒以内に収まるはずです。

カットする箇所

  • 言い間違い

  • 「えーと」「あのー」などの言いよどみ

  • 不自然な間（沈黙）

  • パートとパートの間の余白

カットのタイミング目安（重要）

自然に見えるカットには、フレーム単位の調整が大切です。

カット位置 タイミング 備考
言い始め 発声の直前（0フレーム） ここはギリギリでカット
言い終わり 発声終了から2フレーム後 違和感があれば調整OK

なぜこのタイミング？

  • 言い始めに余白があると、間延びした印象になる

  • 言い終わりをピッタリで切ると、ブツ切り感が出て不自然

  • 2フレームの余韻を残すことで、自然な呼吸感が生まれる

🔍 調整のコツ

言い始めは必ず0フレーム（ギリギリ）でカットしましょう。

言い終わりは2フレームが目安ですが、違和感があれば微調整してください。

ステップ5：テロップ入れ

喋っている内容に合わせてテロップを入れていきます。

テロップの基本設定

  • フォント：基本テロップ１

  • サイズ：100

  • 文字は太く、見やすく設定

  • 配置：画面下部に統一

テロップの入れ方

  1. ツールパネルから「文字ツール」を選択

  2. プログラムモニター上でクリックしてテキストを入力

  3. プロパティパネルでフォント・サイズを調整

  4. テロップの長さを喋っているタイミングに合わせる

強調部分の色変え

重要な部分は色を変えて目立たせます。

  1. 色を変えたい文字をドラッグして選択

  2. プロパティパネルの「塗り」で色を変更

  3. 黄色など、目立つ色を使う

🔍 テロップのコツ

全部の言葉をテロップにする必要はありません。

重要なポイントや強調したい部分を中心に入れましょう。

ステップ6：サイドテロップ作り

画面左側にセクションごとのタイトルを表示するサイドテロップを作ります。

参考動画の「編集領域」のような黒帯＋白枠のデザインです。

サイドテロップの基本設定

  • フォント：ネガティブ６

  • 文字は太く、見やすいサイズに設定

  • 配置：画面左側

セクション一覧

以下のセクションごとにサイドテロップを作成します。

  • はじめに

  • 編集領域

  • 稼働時間

  • ご依頼に関して

  • 最後に

サイドテロップの作り方

  1. ツールパネルから「長方形ツール」を選択

  2. 画面左側に黒い長方形を描画（背景用）

  3. プロパティパネルで「ストローク」にチェックを入れて白枠を追加

  4. ストロークの幅を調整（2〜4px程度）

  5. 「文字ツール」でセクション名を入力

  6. 文字色を白に設定

  7. シェイプの中央にテロップを配置

シェイプとテロップを中央揃えにする

  1. プロパティパネルでテロップとシェイプの両方を選択（Shiftを押しながらクリック）

  2. プロパティパネル →「整列と変形」→「整列」

  3. 「選択範囲に揃える」を選択

  4. 「水平方向中央」「垂直方向中央」をクリック

サイドテロップをコピーして使い回す

1つ目のサイドテロップができたら、コピーして他のセクションに使い回します。

  1. 作成したサイドテロップを選択

  2. Ctrl + C（Mac：Cmd + C）でコピー

  3. Ctrl + V（Mac：Cmd + V）でペースト

  4. テキストをダブルクリックして、セクション名を変更

  5. 該当するセクションのタイミングに配置

🔍 配置のコツ

コピーしたサイドテロップは同じ位置にペーストされます。

横位置はそのままで、タイムライン上で該当セクションに移動させましょう。

ステップ7：シェイプ作り

画面左側に表示する半透明の情報パネルを作ります。

参考動画の名前・ジャンル表示のような白い半透明のパネルです。

シェイプの基本設定

  • 色：白

  • 不透明度：半透明に設定

  • 角丸：あり

  • 枠線：なし

  • 中のテロップ：基本テロップ１

シェイプの作り方

  1. ツールパネルから「長方形ツール」を選択

  2. 画面左側にドラッグして長方形を描画

  3. プロパティパネルの「塗り」をクリックして白を選択

  4. 「不透明度」スライダーで半透明に調整（50〜70%程度）

  5. 「角丸の半径」に数値を入力して角を丸くする

シェイプの中にテロップを入れる

  1. 「文字ツール」で表示したいテキストを入力

  2. フォントを「基本テロップ１」に設定

  3. 文字色を黒に設定（白背景なので）

  4. シェイプの中央に配置する

🔍 配置のコツ

Cmd（Mac）/ Ctrl（Windows）を押しながら選択ツールでシェイプを移動すると、赤いガイド線が表示されます。

この線を目印にして中央揃えにしましょう。

シェイプを位置ズレなくコピーする

同じデザインのシェイプを縦に並べたいとき、横位置を保ったままコピーする方法です。

  1. コピーしたいシェイプを選択

  2. Ctrl + C（Mac：Cmd + C）でコピー

  3. Ctrl + V（Mac：Cmd + V）でペースト（同じ位置に複製されます）

  4. プロパティパネルでY座標の数値だけを変更（X座標はそのまま）

🔍 位置調整のコツ

マウスでドラッグして移動すると、意図せず横にもズレてしまうことがあります。

数値入力で移動させるのが確実です。

ステップ8：アニメーション

テロップやシェイプにアニメーションをつけて、動きのある動画にします。

アニメーションをつける箇所

  • サイドテロップの入場・切り替わり

  • シェイプの入場・切り替わり

  • 基本テロップ（テキストスタイルでパターンをつける）

アニメーションの基本ルール

統一感のある動画にするために、以下のルールを意識しましょう。

同じジャンルの情報は同じアニメーションで統一

例えば「編集スキル」のセクションで、カット・テロップ・画像挿入などを紹介するとき。

これらは同じジャンルの情報なので、同じアニメーション＋同じSEを使って統一感を出します。

サイドテロップは同じアニメーションでOK

「はじめに」「編集領域」「稼働時間」などのサイドテロップは、すべて同じアニメーションで統一しましょう。

基本的に同じアニメーションの使い回しはしない

統一感を出したいとき以外は、同じアニメーションを使い回さないようにしましょう。

バリエーションをつけることで、見ていて飽きない動画になります。

🔍 アニメーションのつけ方

具体的なアニメーションのつけ方は動画で解説しています。

動画を見ながら一緒に作っていきましょう。

ステップ9：BGM

動画全体にBGMを追加します。

BGMの配置

  1. 40番で探したBGMをタイムラインのA2トラックにドラッグ

  2. 動画全体の長さに合わせて配置

フェードイン・フェードアウト

必要に応じて、BGMの最初と最後にフェードをつけましょう。

ステップ10：SE

ポイントとなる箇所に効果音を追加します。

SEをつける箇所

  • サイドテロップの切り替わり

  • シェイプの入場

  • 強調したいポイント

SEの基本ルール

同じジャンルの情報には同じSEを使って統一感を出しましょう。

例：編集スキル紹介（カット、テロップ、画像挿入など）には同じSEを使う。

ステップ11：音量調整

声、BGM、SEのバランスを整えます。

音量の目安

素材 音量目安
声 -6dB
BGM -25dB
SE -18dB

雑音（ザー音）が入っている場合

撮影時の環境音などで「ザー」という雑音が入っていることがあります。

その場合は、オーディオトラックミキサーからクロップフェードをかけて軽減しましょう。

🔍 調整のコツ

声がしっかり聞こえることが最優先です。

BGMは「薄く流れている」くらいがちょうどいいです。

ステップ12：最終確認

書き出す前に、全体を通してチェックしましょう。

確認チェックリスト

  • 動画全体が60秒以内に収まっているか

  • テロップに誤字脱字がないか

  • 音量バランスは適切か（声がしっかり聞こえるか）

  • カットのつなぎが不自然じゃないか

  • アニメーションのタイミングがズレていないか

🔍 確認のコツ

一度最初から最後まで通して再生してみましょう。

気になるところがあればメモして、まとめて修正すると効率的です。

ステップ13：書き出し

確認が終わったら、動画を書き出します。

書き出しの手順

  1. 「ファイル」→「書き出し」→「メディア」を選択

  2. 形式：H.264を選択

  3. プリセット：YouTube 1080p HDを選択

  4. 出力名をクリックして、保存場所とファイル名を設定

  5. 「書き出し」をクリック
